🎯 6 ATTACK SCENARIOS IMPLEMENTED
════════════════════════════════════════════════════════════════════════════════

  [1] CLEAN SMTP (30 samples)
      └─ Direct connection, no obfuscation
      └─ Ground truth: origin IP + country (CN/RU/IN)
      └─ Difficulty: EASY

  [2] VPN-MASKED (30 samples)
      └─ VPN exit IP ≠ true origin IP
      └─ Ground truth: true origin + VPN provider + exit IP
      └─ Difficulty: HARD

  [3] WEBMAIL (30 samples)
      └─ Gmail/Outlook header abuse
      └─ Ground truth: webmail provider + origin IP
      └─ Difficulty: MEDIUM

  [4] PROXY CHAIN (30 samples)
      └─ Multi-hop relay (3+ hops)
      └─ Ground truth: origin IP + all proxy IPs
      └─ Difficulty: HARD

  [5] HEADER FORGERY (30 samples)
      └─ Timestamp regression (Date < Received)
      └─ Ground truth: forged timestamp + real timestamp
      └─ Difficulty: MEDIUM

  [6] TIMEZONE SPOOFING (30 samples)
      └─ Contradictory Date/Received headers
      └─ Ground truth: claimed timezone + actual timezone
      └─ Difficulty: MEDIUM

🧪 VALIDATION HARNESS
════════════════════════════════════════════════════════════════════════════════

huntertrace_harness.py provides:

  1. Run HunterTrace on each synthetic email
  2. Extract predictions (country, obfuscation, confidence)
  3. Compare against ground truth labels
  4. Compute per-scenario accuracy metrics
  5. Generate JSON validation report

  Metrics Tracked:
    • Country attribution accuracy (primary task)
    • Obfuscation detection accuracy
    • Average confidence scores
    • Per-scenario breakdown
